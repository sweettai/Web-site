Novidades

Desenvolvi um site responsivo utilizando HTML, CSS e JavaScript para uma pequena empresa local. Utilizei o Bootstrap para tornar o site mais amigável em dispositivos móveis e implementei algumas funcionalidades simples, como um formulário de contato e uma seção de perguntas frequentes. Aprendi muito durante o processo de desenvolvimento, como trabalhar com Git e GitHub para versionamento de código, além de praticar habilidades de design e desenvolvimento web. Foi uma experiência desafiadora e gratificante, e estou animado para continuar aprendendo e aprimorando minhas habilidades como desenvolvedor.
